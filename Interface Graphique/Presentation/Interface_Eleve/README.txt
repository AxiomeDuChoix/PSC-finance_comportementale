MANUEL D'UTILISATION :

Pour démarrer : exécuter main.py

Pour changer les expériences :
- les expériences sont situées dans le dossier Experiences, et numérotés à partir de 1.
- les numéros des expériences exemples (i.e. avec un nombre d'essai infini) doivent être notées dans le fichier numExemple, séparés par des espaces.
- modifier la valeur de gamma se fait en éditant le fichier gamma dans expériences

RQ : on peut créer une expérience via le générateur proposé dans le dossier parent.
